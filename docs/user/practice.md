# 刷题练习

## 功能简介

c-quiz-app 提供多题库在线练习，支持选择题、判断题、多选题、填空题和解答题。进入练习后题目顺序会打乱，用户可以答题、跳过、查看答案，并在完成后查看得分。

## 如何使用

1. 打开首页，选择要练习的题库。
2. 在题型选择页选择题型，例如选择题、填空题、解答题、全部题目。
3. 进入练习后按页面提示作答。
4. 不想作答时点击“跳过此题”。
5. 填空题可点击“查看答案”，但仍可继续输入并提交。
6. 到最后一题后进入练习完成页，查看得分。
7. 如果首页有错题数量徽标，可进入错题本练习。

## 题型说明

| 题型 | 操作 |
| --- | --- |
| 选择题 / 判断题 | 点击选项作答，首次答对计分，答错会进入错题记录 |
| 多选题 | 勾选多个选项后提交 |
| 填空题 | 一个空对应一个输入框；按 Enter 会跳到下一个空或提交 |
| 解答题 | 输入文字答案后提交，系统展示参考答案和得分 |

## 错题本

- 错题按题库存储在本地浏览器。
- 首页题库卡片会显示错题数量。
- 进入错题本练习后，会读取本地错题并清空该题库的错题徽标。

## 在线聊天

首页和练习页右下角有聊天入口：

1. 首次进入会提示输入昵称，留空则使用匿名用户。
2. 打开聊天面板后可以发送消息、使用快捷 emoji。
3. 点击在线用户名称可以快速 `@` 对方。
4. 普通消息会以弹幕形式飘过页面，不会阻挡答题操作。

## 排行榜

练习完成后，系统会把当前昵称和得分上报到本周排行榜。首页排行榜会定时刷新，展示 Top 10。

## 常见问题

Q: 为什么每次题目顺序不一样？  
A: 练习开始时会打乱题目顺序，这是为了减少机械记忆。

Q: 查看答案后还能提交吗？  
A: 填空题可以。查看答案只是展示参考答案，不会立即结束本题。

Q: 为什么有的题目图片没有显示？  
A: 可能是题库资源路径缺失或图片文件未同步，需要管理员检查题库资源。

Q: 聊天昵称怎么改？  
A: 昵称保存在浏览器本地存储中。当前界面没有专门修改入口，可清除浏览器本地数据后重新输入。
