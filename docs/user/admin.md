# 管理员功能

## 功能简介

管理员页面提供两个主要功能：访客日志和题库管理。访客日志用于查看访问设备、真实 IP、来源地、在线情况和封禁状态；题库管理用于创建题库、修改题库信息、查看题目、删除题目和批量导入 JSON。

## 前置条件

- 需要管理员密码。
- 访问入口在首页右上区域的“管理员日志”，或直接访问 `#admin`。

## 访客日志

1. 打开管理员页面。
2. 选择“访客日志”。
3. 输入管理员密码并登录。
4. 查看最近访客记录。
5. 如需限制访问，先填写封禁原因，再点击“封禁设备”或“封禁 IP”。
6. 在封禁列表中可以查看生效和历史封禁记录，也可以执行解封。

访客表格包含：

| 列 | 含义 |
| --- | --- |
| 设备 | 设备摘要和设备 ID |
| 网络位置 | IP 和来源地 |
| 活跃情况 | 最近活跃、首次访问、打开次数、最后页面 |
| 封禁状态 | 正常、设备已封禁、IP 已封禁 |
| 操作 | 封禁或解封 |

## 题库管理

1. 打开管理员页面。
2. 选择“题库管理”。
3. 输入管理员密码并登录。
4. 在题库列表中点击题库卡片查看题目。
5. 可以创建新题库、编辑题库名称/图标/排序、删除题库。
6. 可以打开导入窗口，用 JSON 批量替换该题库下的题目。
7. 可以在题目列表中删除单题。

## 导入题目

导入内容必须是包含 `questions` 数组的 JSON：

```json
{
  "questions": [
    {
      "序号": 1,
      "题目类型": "选择题",
      "题目内容": "题干",
      "选项": { "A": "选项 A", "B": "选项 B" },
      "正确答案": "A",
      "答案文本": "选项 A"
    }
  ]
}
```

导入会替换当前题库下的题目。执行前请确认选择的是正确题库。

## 常见问题

Q: 设备封禁和 IP 封禁有什么区别？  
A: 设备封禁基于系统生成的设备 ID，IP 封禁基于访问 IP。设备封禁更适合限制同一浏览器设备，IP 封禁更适合限制同一网络来源。

Q: 删除题库会发生什么？  
A: 删除题库会连同该题库下的题目一起删除。执行前页面会弹确认框。

Q: 导入 JSON 后题目数量不对怎么办？  
A: 检查 JSON 顶层是否包含 `questions` 数组，并确认题型、答案字段符合格式要求。

Q: 首页题库数量和数据库数量不一致怎么办？  
A: 首页部分卡片文案可能是静态展示，真实题库数据以 `/api/quiz/banks` 和管理端题库列表为准。
