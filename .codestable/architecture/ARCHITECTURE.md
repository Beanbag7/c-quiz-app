## 项目概述

**项目名**：c-quiz-app  
**技术栈**：React + Vite，前端单页应用  
**用途**：题库问答 Web 应用，支持刷题练习

## 目录结构

```
src/
├── App.jsx           主应用组件
├── components/       业务组件
├── assets/           静态资源
QuestionData/         题库 JSON 数据
scripts/              脚本工具（题库爬取等）
docs/archive/         历史文档归档
```
