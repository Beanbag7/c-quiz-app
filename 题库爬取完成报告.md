## 🔧 技术实现

### 使用的API端点

1. **登录API**
   ```
   POST https://ptougeapi.keepctf.cn/api/login
   参数: {"account": "username", "password": "password"}
   ```

2. **获取题目API**
   ```
   POST https://ptougeapi.keepctf.cn/api/classroom/exam/show
   参数: {"id": exam_id, "classroomId": classroom_id}
   ```

3. **获取答案API** ⭐ 关键发现
   ```
   POST https://ptougeapi.keepctf.cn/api/classroom/exam/getTmp
   参数: {"id": exam_id, "classroomId": classroom_id}
   注意：使用 "id" 而不是 "paperId"！
   ```

### 关键发现

**问题**：最初使用 `paperId` 参数调用答案API时，返回错误"请选择操作记录"

**解决**：经过测试发现，应该使用 `id` 而不是 `paperId` 作为参数名

**正确的参数格式**：
```json
{
  "id": 3922,
  "classroomId": 349
}
```

## 📝 答案数据格式

API返回的答案数据结构：
```json
{
  "questionId": 96708,
  "answer": "用do-while语句构成的循环，在while后的表达式为零时结束循环",
  "isTrue": 1
}
```

- `questionId`: 题目ID（用于与题目配对）
- `answer`: 答案内容
- `isTrue`: 是否正确（1=正确，0=错误）
